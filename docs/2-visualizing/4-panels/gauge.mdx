---
title: Gauge
description: Display numeric values from an incoming topic message path on a customized gauge display.
---

Display numeric values, or string values that can be converted to numeric values, from an incoming topic message path on a customized gauge display.

![gauge panel](/img/docs/visualizing/panels/gauge/panel.webp)

## Settings

| General        |                                                                                   |
| -------------- | --------------------------------------------------------------------------------- |
| **Data**       | Topic message path for the displayed numeric value                                |
| **Minimum**    | Minimum value for gauge display                                                   |
| **Maximum**    | Maximum value for gauge display                                                   |
| **Color mode** | Coloring logic for gauge display ("Color map" or "Gradient")                      |
| **Color map**  | Predetermined color maps for gauge display: "Red to green", "Rainbow", or "Turbo" |
| **Gradient**   | Two colors for creating a gradient across gauge display                           |
| **Reverse**    | Reverse the color map or gradient across gauge display                            |
