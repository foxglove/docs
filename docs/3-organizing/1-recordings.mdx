---
title: Recordings
---

Recordings represent every data file recorded by your organization's devices.

## Import recording

Import a data recording for a device in your organization.

|        |                                                                 |
| ------ | --------------------------------------------------------------- |
| In app | Import at [Recordings](https://console.foxglove.dev/recordings) |
| CLI    | `$ foxglove TBD`                                                |

Check out Foxglove's [supported data formats](TBD) for import.

## Delete recording

Delete a data recording imported into your organization.

|        |                                                                           |
| ------ | ------------------------------------------------------------------------- |
| In app | Select recording at [Recordings](https://console.foxglove.dev/recordings) |
| CLI    | `$ foxglove TBD`                                                          |

## Attach information to recording

To include contextual information in a recording, import [MCAP](https://mcap.dev) files with [metadata records](TBD).

To include auxiliary information in a recording, import [MCAP](https://mcap.dev) files with [attachments](TBD).
