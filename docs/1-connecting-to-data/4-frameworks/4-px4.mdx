---
title: PX4
description: Foxglove can load local PX4 ULog (`.ulg`) files.
---

To load a local [PX4 ULog](https://docs.px4.io/main/en/dev_log/ulog_file_format.html) (`.ulg`) file, double-click it from your file manager, drag-and-drop the file directly into the app, or _Open local file_ via the _Data source_ dialog:

![Local file dialog](/img/docs/connecting-to-data/local-file.png)
