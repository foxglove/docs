Display numeric values from an incoming topic message path on a customized gauge display.

![gauge panel](/img/docs/visualization/panels/gauge/panel.webp)

## Settings

### General

| field          | description                                                                                      |
| -------------- | ------------------------------------------------------------------------------------------------ |
| **Data**       | Topic message path to a numeric value (or string value that can be converted to a numeric value) |
| **Minimum**    | Minimum value for gauge display                                                                  |
| **Maximum**    | Maximum value for gauge display                                                                  |
| **Color mode** | Logic for coloring gauge display ("Color map" or "Gradient")                                     |
| **Color map**  | Predetermined color maps for gauge display: "Red to green", "Rainbow", or "Turbo"                |
| **Gradient**   | Two colors for creating a gradient across gauge display                                          |
| **Reverse**    | Reverse the color map or gradient across gauge display                                           |
